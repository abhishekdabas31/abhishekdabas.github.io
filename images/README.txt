# Image Placeholders

This folder should contain the following images for your portfolio website:

1. profile-placeholder.jpg - A professional headshot (recommended size: 500x500px)
2. project1.jpg - Image for your first project (recommended size: 800x600px)
3. project2.jpg - Image for your second project (recommended size: 800x600px)
4. project3.jpg - Image for your third project (recommended size: 800x600px)
4. project4.jpg - Image for your fourth project (recommended size: 800x600px)

When you don't have actual images, the website will automatically generate colored placeholders with code icons.

## Recommended Image Sources:

1. Your own professional photos
2. Screenshots of your actual projects
3. Free stock photos from:
   - Unsplash (https://unsplash.com)
   - Pexels (https://www.pexels.com)
   - Pixabay (https://pixabay.com)

## Image Optimization:

For better performance, consider optimizing your images before using them:
- Resize to appropriate dimensions
- Compress to reduce file size (using tools like TinyPNG or ImageOptim)
- Consider using WebP format for better compression
